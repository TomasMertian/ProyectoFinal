# Proyecto Final Integrador

Este proyecto final, desarrollado para la asignatura Arquitectura y Sistemas Operativos, consiste en:

- Consumir datos de un dispositivo IoT a través de un endpoint público.
- Procesarlos en tiempo real con Node-RED.
- Visualizarlos en contenedores Docker.

La solución incluye un flujo que consulta datos periódicamente, extrae los dos registros más recientes y los muestra mediante widgets, demostrando el uso práctico de tecnologías como Docker, APIs REST y sistemas IoT.

## Cómo correr el proyecto

1. Asegúrate de tener Docker instalado en tu sistema.
2. Navega a la carpeta del proyecto que contiene el archivo `docker-compose.yml`.
3. Ejecuta el siguiente comando en la terminal: docker-compose up
4. Accede a la interfaz de Node-RED en `http://localhost:1880` para visualizar el flujo y los datos.
